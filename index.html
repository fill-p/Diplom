  • About
    Программа створена для вивчання JavaScript. Для цього в програмі існують теоретичні матеріали та практичні завдання, для закріплення теоретичних відомостей. Якщо учень не може виконати завдання, то існують відповіді, які допоможуть по коду-прикладу, виконати завдання іншим способом. Також для більш глибокого вивчення мови, програмою передбачено створення інтерфейсів за допомогою Html та CSS, наприклад для вивчення DOM, або взаємодії з кнопками.

  • Help
    1. Для роботи в програмі потрібно вписати в текстове поле код на мові Html/CSS/JS та запустити його за допомогою однієї з кнопок вгорі;
    2. При наведенні на кнопки:

    висвічується підказка, яка пояснює функціонал кнопок;
    3. В блоці 'Уроки' знаходяться завдання. При одиничному натискані на урок - текстове поле заповниться завданням та початковими умовами. При подвійному кліку по уроку - текстове поле заповниться відповіддю;
    4. В текстовому полі працюють більшість комбінацій клавіш, такі як:
    Ctrl + C - копіювати
    Ctrl + V - вставити
    Ctrl + X - вирізати
    Ctrl + S - зберегти
    Ctrl + F - пошук
    Ctrl + Space - підказки(працюють лише в javascript коді)
    Ctrl + Z - скасовує останню зміну
    Ctrl + Y - повторює останню зміну
